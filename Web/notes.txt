**********
<input type="text" name="phone_number">
That name attribute is not how the server stores your data. 
That's how the browser packages and labels the data before sending it.
 The server receives a message that says "phone_number = 0241234567" 
 but what the server does with it, where it puts it, whether it saves it to a database
  or throws it away — that is entirely decided by your Python code on the server side.
